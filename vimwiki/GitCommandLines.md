Git is a distributed VCS (Version Control System)

git config --global user.name "your name"
git config --global user.email "your email.com"

git config --list -> lists configuration

git help config -> list manual page for the config action
git config --help

git init -> local code base you want to start tracking with git

git branch Branch_Name_Here
git checkout Branch_Name
git status
git add -A
git push -u origin Branch_Name
